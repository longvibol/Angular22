Key word to declare variable 

1-var : is the old one : бһ–бҹҒбһӣбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ function then we can called it outside. 
бһ…бһ№бһ„бһңбһ¶бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһҳбһ¶бһ“ bug бһҖбһҫбһҸбһЎбһҫбһ„

==> it has function scope, not bloc scope 

2-let : we can declare it instead of var 

бһ–бҹҒбһӣбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ function бһ бһҫбһҷбһҷбһҫбһ„бһҳбһ·бһ“бһўбһ¶бһ…бһҷбһҖбһңбһ¶бһҳбһҖбһ”бҹ’бһҡбһҫбһ“бҹ… бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ”бһ¶бһ“бһ‘бҹҖбһҸбһ‘бҹҒ

==> block-level scope 

3-const : after we assign we can not reassign it 

бһӮбҹҒбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ API Called бһ–бҹ’бһҡбҹ„бҹҮбһ–бҹҒбһӣбһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһңбһ¶бһ бһҫбһҷ бһҷбһҫбһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҸбһјбһҡбһңбһ¶бһ”бһ¶бһ“бһ‘бҹҒ

бһҮбһ¶ Default бһҷбһҫбһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ const бһ бһҫбһҷбһңбһ¶бһҷбһҫбһ„бһ…бһ„бҹӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһҸбһ¶бһҳбһҖбҹ’бһҡбҹ„бһҷбһҷбһҫбһ„ бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ let

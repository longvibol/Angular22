Key word to declare variable 

1-var : is the old one : ពេលយើងប្រើប្រាស់វានៅក្នុង function then we can called it outside. 
ចឹងវាធ្វើអោយមាន bug កើតឡើង

==> it has function scope, not bloc scope 

2-let : we can declare it instead of var 

ពេលយើងប្រើវានៅក្នុង function ហើយយើងមិនអាចយកវាមកប្រើនៅ ខាងក្រៅបានទៀតទេ

==> block-level scope 

3-const : after we assign we can not reassign it 

គេប្រើប្រាស់វាជាមួយ API Called ព្រោះពេលយើងប្រើវាហើយ យើងមិនអាចប្តូរវាបានទេ

ជា Default យើងប្រើប្រាស់ const ហើយវាយើងចង់ធ្វើការផ្លាស់ប្តូរតម្លៃតាមក្រោយយើង ប្រើប្រាស់ let

==> never use Var in the modern typescript code (X)

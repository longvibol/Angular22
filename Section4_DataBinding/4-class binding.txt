- Add or remove the class 

CSS:

  background-color: lightgreen;
  color: darkgreen;
  padding: 10px;
}

.danger {
  background-color: lightpink;
  color: darkred;

}

HTML:

 <div class="metric-card" [class]="salesValue >= targetValue ? 
   'success' : 'danger'">
    <div class="label">Current Sales</div>
    
 ==> In Angular we can use shortcut way to create class binding we no need to use property binding 
 
 [class.success]="salesValue >= targetValue" 
 [class.danger]="salesValue < targetValue "

 Using Method to display classes : 
 
 IN TS:
 
 // class binding method 
  getPerformanceClass(): string{
      const percentage =(this.salesValue / this.targetValue) * 100;
      if(percentage < 50)
        return 'danger';
      else if (percentage < 80)
        return 'warning';
      else
        return 'success';
    }
  
 IN HTML:
 
 Cancantinate the class (string connect) 
 <div [class]="getPerformanceClass() + 'metric-card'">
 
 
 this is better than style (style is the inline style difficult to manintance)
